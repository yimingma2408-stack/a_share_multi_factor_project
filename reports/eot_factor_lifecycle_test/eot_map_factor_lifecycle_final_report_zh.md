# 基于 EOT-map 两样本检验的 A 股因子生命周期诊断

## 1. 执行摘要

本项目将原先未经尺度调整、基于距离的 EOT 漂移诊断，升级为正式的共同参考点 EOT-map 统计检验。新框架加入了中心化 IID 与分块乘子校准、坐标贡献分解、有符号恶化诊断，以及由统计检验驱动的生命周期状态。最终面板包含 3,165 个“因子－周度监控时点”检验和 9,495 条坐标诊断记录；每个监控时点分别进行了 300 次 IID 和 300 次分块乘子抽样。换手率与波动率因子最频繁地拒绝分布同质性。下行收益既是最常见的最大变化坐标（2,181/3,165），也是最常见的主导恶化坐标（1,564/3,165）。

本框架最明确的价值是**监控，而不是配置增益**。在 10 bps 成本假设下，合资格因子等权策略的样本 Sharpe 最高，为 0.022；所有正式检验驱动的加权策略都更低。在相关性零假设的合成实验中，分块乘子将拒绝率从 60% 降至 40%，但仍显著高于名义水平。只要这些限制得到明确披露，本项目适合作为透明的研究与工程作品；但不能据此声称获得了配置 Alpha 或可直接实盘交易。

### 核心证据一览

| 证据 | 结果 | 含义 |
| --- | ---: | --- |
| 因子－日期 EOT 检验 | 3,165 | 对十个因子进行了滚动监控 |
| 坐标诊断记录 | 9,495 | 每次检验包含三个表现坐标 |
| 分块 bootstrap 原始拒绝率 | 27.1% | 预警数量高于 IID 基准 |
| 跨因子 FDR 拒绝率 | 23.4% | 通过同日多重检验控制的预警 |
| 最常见的主导变化 | 下行收益，共 2,181 次 | 尾部表现驱动了多数分布变化 |
| 10 bps 下最佳策略 | 等权，Sharpe 0.022 | 检验驱动的加权没有增加配置价值 |

## 2. 与旧版本的区别

旧版输出是基础窗口与近期窗口两个重心映射之间的平均平方距离。它既不是 Sinkhorn 运输成本，也不是经过校准的统计检验。新实现保留了该未缩放距离，同时加入 `nm/(n+m)` 有效样本量缩放，并将缩放后的统计量与中心化 bootstrap 零分布比较。原始 p 值与 FDR 调整后的 q 值取代了仅依据 `eot_drift_zscore` 的解释方式；旧版漂移结果仍保留作为基准，不被覆盖。

## 3. 方法

对于因子 `j` 和监控日期 `t`，周度表现向量定义为：

```text
Z = (Rank IC, 多空收益, 下行收益)
```

基础样本包含 `t-182` 至 `t-27` 周，近期样本包含 `t-26` 至 `t-1` 周。中位数和 MAD 只由基础窗口估计，并同时用于转换两个样本。每次检验使用独立生成、随后固定不变的单位球参考点。基础与近期经验分布都从同一组参考点进行映射，共享的 epsilon 等于标准化合并样本非零平方距离中位数的 0.2 倍。

符号约定为：

```text
D = T_recent - T_base
```

正式统计量为：

\[
\mathcal T_{n,m}=\frac{nm}{n+m}\frac1N\sum_l\|D(U_l)\|^2.
\]

| 设计项 | 主规格 |
| --- | --- |
| 表现向量 | `（Rank IC，多空收益，下行收益）` |
| 基础/近期窗口 | 156/26 周，均严格早于监控日期 |
| 标准化 | 基础窗口中位数和 `1.4826 × MAD` |
| 参考分布 | 从三维单位球固定抽取 100 个参考点 |
| 熵正则化 | `0.2 ×` 合并样本非零平方距离中位数 |
| 正式统计量 | 有效样本量 × 共同参考点映射平均平方差 |
| 校准 | 300 次 IID 和 300 次八周分块乘子抽样 |
| 多重检验 | 同一日期跨因子 10% BH-FDR |
| 持续预警 | 最近三个监控时点中至少两次 FDR 拒绝 |

## 4. Bootstrap 校准

IID 指数权重用于重新生成加权经验映射。零分布抽样使用中心化映射增量，而不是直接计算两个 bootstrap 映射之间的距离。分块扩展让连续八周共享同一个乘子。IID 是与独立样本理论最一致的基准；分块方法只是针对时间依赖的探索性校准。每个场景 20 次重复的合成实验显示，存在时间依赖时检验尺寸明显偏大，因此尚不能声称得到了普遍有效的尺寸控制。更多细节见 [`bootstrap_calibration_report.md`](bootstrap_calibration_report.md)。

## 5. 全局检验结果

分块校准下，原始拒绝率最高的因子依次为：`turnover_1m`（33.2%）、`volatility_3m`（33.2%）、`volatility_1m`（31.7%）、`turnover_3m`（31.6%）和 `momentum_3m`（29.4%）。跨因子 FDR 拒绝占全部面板记录的 23.4%；满足滚动“三期中至少两次”持续预警规则的记录同样占 23.4%。IID 原始拒绝率为 19.1%，低于分块方法的 27.1%。这些结果是样本内校准诊断，不能直接推广为总体结论。

## 6. 坐标诊断

在 3,165 次检验中，下行收益在 2,181 次检验里贡献了最大的平方变化，多空收益为 787 次，Rank IC 为 197 次。若只观察恶化方向，主导恶化指标为下行收益的有 1,564 次，多空收益为 876 次，Rank IC 为 725 次。“最大变化贡献”与“最大恶化贡献”被有意区分；两者都只是描述性归因，不是因果归因。

| 坐标 | 最大变化次数 | 主导恶化次数 |
| --- | ---: | ---: |
| Rank IC | 197 | 725 |
| 多空收益 | 787 | 876 |
| 下行收益 | 2,181 | 1,564 |

![坐标变化与恶化次数](figures/16_coordinate_diagnostic_counts.png)

这种区别具有实际意义：Rank IC 很少是平方变化最大的坐标，却在 725 次检验中成为主导恶化指标。一个指标对总体位移的贡献可能较小，但仍可能提供最强的负向证据。

## 7. 生命周期状态

面板共识别出 1,126 条 `Healthy`、950 条 `Watch`、220 条 `Decaying`、364 条 `Recovering` 和 505 条 `Dormant` 记录。状态规则综合使用严格过去信息下的历史/近期 ICIR、质量趋势、FDR 拒绝、持续预警和核心坐标的有符号恶化。每一行都保存了分类原因，因此比旧版仅依赖平滑距离阈值的规则更具可解释性。

## 8. 监控价值

该检验能够识别因子预测排序关系、多空收益和下行行为的联合变化，并指出哪个坐标主导了变化。持续性规则和 FDR 控制降低了对孤立原始 p 值的依赖。目前最有依据的用途，是对因子健康状态进行当期监控。

## 9. 配置价值

七种 walk-forward 方案比较了因子等权、ICIR、旧版距离惩罚、正式显著性惩罚、有符号恶化惩罚和两种生命周期过滤策略，并测试了 0/5/10/20 bps 成本。在 10 bps 下，等权策略以 0.022 的 Sharpe 居首；旧版距离惩罚为 -0.157，正式显著性惩罚为 -0.170，正式有符号惩罚为 -0.173。因此，正式校准提高了可解释性，但没有在本样本中表现出配置增益或成本稳健的 Alpha。

| 10 bps 下的策略 | 年化收益 | Sharpe | 最大回撤 |
| --- | ---: | ---: | ---: |
| 合资格因子等权 | 0.04% | 0.022 | -3.54% |
| 旧版距离 EOT | -0.65% | -0.157 | -11.27% |
| ICIR 加权 | -0.65% | -0.157 | -11.34% |
| 正式显著性惩罚 | -0.70% | -0.170 | -11.50% |
| 正式有符号惩罚 | -0.72% | -0.173 | -11.49% |
| 保守生命周期过滤 | -0.89% | -0.224 | -12.04% |
| 生命周期过滤 | -1.41% | -0.355 | -15.23% |

![10 bps 成本下的策略 Sharpe](figures/18_strategy_sharpe_10bps.png)

配置对比给出的经济结论非常清楚，而不仅仅是“统计上不显著”：在 10 bps 下，每一种自适应加权方案都落后于等权基准。因此，EOT-map 层的合理定位是监控与归因工具，而不是已经得到验证的 Alpha 来源。

## 10. 局限性

1. 周频因子表现存在序列相关性。
2. IID bootstrap 理论不能自动覆盖该时间序列场景。
3. 分块乘子是应用性扩展，目前缺少匹配的理论证明。
4. 滚动窗口高度重叠。
5. 多个因子和多个日期带来的多重性，超过了单日横截面 BH 的控制范围。
6. 坐标贡献不是因果归因。
7. 各项表现指标包含估计误差。
8. 历史结果不能保证实盘表现。
9. 对 epsilon 和参考点数量的稳健性检验仍然有限。
10. 财务因子仍需要覆盖广泛且满足 Point-in-Time 要求的数据。
11. 合成实验每个场景只有 20 次重复，无法精确估计尾部检验尺寸。
12. 即使使用分块校准，AR(1) 合成零假设下的拒绝率仍然明显偏高。

## 11. 项目最终定位

建议项目标题为：**基于 EOT-map 两样本检验的 A 股因子生命周期诊断**。可以将**实验性漂移感知多因子加权**作为次要扩展，但不应将其作为核心成果。

## 12. 简历与面试表述

### 简历要点

- 构建共同参考分布上的正式 EOT-map 两样本检验，以中心化乘子 bootstrap 校准 A 股因子生命周期变化。
- 将联合统计量分解为 Rank IC、多空收益与下行收益坐标贡献，并通过有符号诊断区分改善与恶化。
- 实现跨因子 FDR、持续预警、生命周期监控及实验性保守调权，明确区分监控证据与无法验证的实盘收益主张。

### 30 秒介绍

我将一个因子漂移原型升级为经过校准的 EOT-map 检验系统。系统使用共同参考点映射、中心化 bootstrap p 值和坐标级方向诊断，比较过去三年的周频因子表现与最近六个月表现。项目最明确的结果是提升了因子监控的可解释性；权重调整仍属于实验性扩展，而且没有战胜等权基准。

### 2 分钟介绍

项目从十个 A 股因子的周频 Rank IC、多空收益和下行收益出发。标准化参数只使用历史窗口估计，基础样本和近期样本从相同的独立单位球参考点进行映射，并按有效样本量缩放统计量，再使用中心化 IID 指数乘子进行校准。考虑到周频数据存在时间依赖，我还实现了八周分块乘子扩展，但将其明确标记为探索性方法。全局 p 值经过跨因子 BH 校正和“三期中至少两次”的持续预警规则；映射差异进一步按坐标拆解并赋予方向，因此生命周期状态能够解释变化属于改善还是恶化。最后，我比较了七种严格使用过去信息的月度配置方案并计入交易成本。结果表明，正式检验提高了监控的可解释性，但没有证明存在成本稳健的配置 Alpha，也不能作为实盘策略依据。
